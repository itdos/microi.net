# 平台数据库表说明

> 本文档记录 Microi吾码 平台各模块的数据库表结构说明，方便开发者了解底层数据模型。

## 工作流引擎（WF）

工作流引擎相关表，用于支撑流程设计、流程发起、待办管理及流程轨迹记录等功能。

---

### WF_FlowDesign - 流程图设计表

> 存储流程图的设计信息，一个流程图对应一条数据。


### WF_Node - 流程节点属性表

> 存储流程图中每个节点的配置属性，如审批人、超时策略等。



### WF_Line - 流程条件（线）属性表

> 存储流程图中连线（条件）的配置属性，用于控制流程流转的条件判断。



### WF_Flow - 流程实例表

> 存储每次发起的流程实例，发起一个流程生成一条记录，对应 `WF_Work` 表中 N 条工作数据。



### WF_Work - 流程工作待办表

> 存储流程产生的每一条待办任务，一个流程实例可产生 N 条待办记录（如并行审批时同时产生多条）。



### WF_History - 流程轨迹表

> 详细记录流程每一步、每个人的所有操作，如同意、拒绝、撤回等，用于流程审计和进度查看。
