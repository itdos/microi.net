# Successful Case Study of Group Manufacturing Collaboration System Based on Microi Code
## Group Manufacturing Collaborative System
* Products of A-share listed companies
* The development cycle is 3 months+, developed by a customer R&D team of more than 10 people, with technical support provided by the official team of Microi Wuma
## Introduction and Highlights of the Group Manufacturing Collaborative System
* Over 200 functional modules, 241 physical tables, and 339 interface engines have been configured
* Docking with Feishu, processing logical processes and message notifications on Feishu
* Microi Code Platform expands 11 third-party system databases within the group
* Multi database and multi table statistics for virtual reports, and implementation of business logic for adding, deleting, and modifying virtual reports
