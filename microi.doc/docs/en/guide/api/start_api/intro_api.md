# Introduction
## Introduction
**Microi Code** The back end is based on`.NET9、SqlSugar、Redis、Dos.ORM、RabbitMQ、ES`The latest technology stack development. through middleware`ORM`Realize cross-library read and write, support multiple data sources, support`MySql5.5+、SqlServer2016+、Oracle11g+`, supports read/write separation/sub-database and sub-table, which can be extended to more database types. Support for distributed deployment, available through`Docker、K8S、Jenkins、Rancher、CICD`and other tools to automate deployment. Support single sign-on SSO, built-in task scheduling engine, support distributed task scheduling.
