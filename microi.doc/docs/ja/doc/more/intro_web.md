# 紹介
## プロフィール
**Microi吾コーデ** フロントエンドプロジェクトは`Vue2/3 + Element-UI/Element-Plus`フレーム作り。`Vue2`バージョンは現在すべてオープンソースになっており、複数のバージョンの反復とN以上の現実的なプロジェクトの実戦を経て、現在すでに複数の生産環境で安定して運行している。迅速な開発、保守、拡張が可能なフロントエンドプロジェクトです。`Vue3`バージョンは現在も着実に開発されており、新しいバージョンのUIは採用されている`macos`スタイル、お楽しみに。



## Vue2版で身につけるべき基础知识

- このプロジェクトは、Vue2.x、webpack4.x、vue-router3.x、vuex3.x、element-uiに基づいて開発された。

- 開発する前に以下のことを勉強して、これらの知識を事前に理解して勉強することをお勧めします。
	- [Vue2ドキュメント](https://v2.cn.vuejs.org/)
	- [Vue-Routerドキュメント](https://router.vuejs.org/zh/guide/)
	- [Webパックドキュメント](https://webpack.docschina.org/)
	- [TypeScriptドキュメント](https://www.typescriptlang.org/zh/docs/)
	- [Vuexドキュメント](https://v3.vuex.vuejs.org/zh/guide/)
