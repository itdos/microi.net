# 紹介
## プロフィール
**Microi吾コーデ** バックエンドは`.NET9、SqlSugar、Redis、Dos.ORM、RabbitMQ、ES`最新の技術スタック開発を待っています。ミドルウェア経由`ORM`ライブラリ間の読み書きを実現し、複数のデータソースをサポートし、サポートします`MySql5.5+、SqlServer2016+、Oracle11g+`、読み書き分離/ライブラリ分割テーブルをサポートし、より多くのデータベースタイプを拡張できます。分散配置をサポートします。`Docker、K8S、Jenkins、Rancher、CICD`などのツールが自動導入を実現します。シングルサインオンSSOをサポートし、タスクスケジューリングエンジンを内蔵し、分散タスクスケジューリングをサポートします。
