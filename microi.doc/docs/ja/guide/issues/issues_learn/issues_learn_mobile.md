# ユニ-appチュートリアル

## 紹介

「Uni-app」は「vue.Js」を使ってすべてのフロントエンドアプリケーションを開発するフレームワークで、開発者はコードを作成しiOS、Android、Web (応答式)、各種のアプレット (マイクロレター/支付宝/百度/トップ/飛書/QQ/速手/ホッチキス/淘宝)、クイックアプリケーションに公開できますなど複数のプラットフォーム。

「ユニ-app」は開発者の数、ケース、端を越えて平らにし、柔軟性、性能体験、周辺生態、学習コスト、開発コストなどの8つの重要な指標でより強い優位性を持っている。

公式サイト住所:[https://uniapp.dcloud.net.cn/](https://uniapp.dcloud.net.cn/)
 
 
![uniapp](/issues/uniapp01.jpeg)

## 機能フレーム図
次の「ユニ-app」機能フレーム図から、「ユニ-app」はクロスプラットフォームの過程で、プラットフォームの特色を犠牲にすることなく、プラットフォームの独自能力を優雅に呼び出すことができ、本当に海納百川、各所長を実現する。
