# # # Microi吾コードに基づく成功事例-標準CRM製品
## 標準CRM製品
* 開発サイクルは1ヶ月で、お客様の研究開発チームの私コード公式チームが共同で開発します
## ハイライト
* 200機能モジュール、252枚の物理テーブル、76個のインタフェースエンジン、29個のワークフローエンジンが構成されています
* 低コードプラットフォーム標準H5モバイル端末 (PC側構成後にモバイル端末が同期して有効)
