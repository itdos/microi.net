# Microi吾コードに基づく成功事例-インターネット不動産中古住宅、新築住宅、賃貸住宅プラットフォーム
## インターネット不動産中古住宅、新築住宅、賃貸住宅プラットフォーム
* 開発サイクルは3ヶ月で、開発者は5人以上で、Microi吾コード公式チームが開発した
## インターネット不動産の紹介、ハイライト
* ピーク時400線下店、2000軒の仲介人、本部100人
* 100以上のメニューモジュール、128枚の物理テーブルを設定します
* 大量のフロントエンドマイクロサービスのカスタム開発、バックエンド二開、フォームエンジンはvueカスタムコンポーネント、カスタムコンポーネントを埋め込み、フォームエンジンを二次呼び出します
* 阿里雲プライバシー番号 (仲介人がプライバシー番号を結び付けて電話マーケティングを行う) をドッキングし、プラットフォームは通話音声の内容を記録した
* VRデバイスをドッキングしました
* 完全なPC側公式サイト、プラットフォーム管理システム、携帯電話側ios app、アンドロイドapp、マイクロメッセンジャー、h5
* 地図探し、地下鉄探し、画輪探しを実現しました。
