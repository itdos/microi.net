# Microi吾コードによる成功事例-グループ製造協同システム
## グループ製造協同システム
* A株上場会社の製品
* 開発サイクルは3ヶ月で、お客様の研究開発チームが10人以上開発し、Microi吾コード公式チームが技術サポートを提供しています
## グループ製造協同システムの紹介、ハイライト
* 200機能モジュール、241枚の物理テーブル、339個のインタフェースエンジンが構成されています
* 飛書をドッキングし、飛書で論理的な流れ、メッセージ通知を処理する
* Microi吾コードプラットフォームはグループ内部の11の第三者システムデータベースを拡張します。
* 複数のデータベース、複数のテーブルが仮想レポートを統計し、仮想レポートの追加、削除、ビジネスロジックの変更を実現します
