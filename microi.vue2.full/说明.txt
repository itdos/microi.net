使项目跑起来（使用nodejs 14.x）：
# nvm use 14
# npm install nrm -g
# nrm use taobao
# npm install
# npm run dev 

）若以上步骤出现其它错误，可以尝试下以下步骤：
    ）删除node_modules
    ）删除package-lock.json
    ）执行# npm cache clean --force
    ）重新执行#npm install安装环境步骤
